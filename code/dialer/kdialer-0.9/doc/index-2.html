2. Installation

2.1 How to obtain KDialer

KDialer can be downloaded from the author's homepage: http://www.xs4all.nl/~icehawk/ Here, you can also view some screenshots of KDialer and other projects.
KDialer can also be downloaded directly at http://www.xs4all.nl/~icehawk/linux/kdialer-0.9.tar.gz

2.2 Requirements

In order to successfully compile KDialer, you need the latest versions of libkdecore and Qt. The KDE libraries can be found on ftp://ftp.kde.org/pub/kde/. Qt can be downloaded from http://www.troll.no, but probably you already have that.
And ofcourse you'll need a modem duhh...

2.3 Compilation and installation

In order to compile and install KDialer on your system, type the following in the base directory of the KDialer distribution: 

# ./install.sh

Since the package has not completely been finished yet, it still uses a custom build install script. The final version will use a Makefile.
Should you run into problems please report them to the author at icehawk@xs4all.nl.
