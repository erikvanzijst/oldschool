5. Questions and answers

5.1 KDialer doesn't work

If KDialer refuses to dial after a normal installation, check whether:

1. you're modem is turned on;
2. there isn't another program using the modem;
3. you specified the correct modem device*.

*On Linux 2.1.XXX kernels you should avoid the use of /dev/cua? devices. If you use a developers kernel, always be sure to use /dev/ttyS? instead.
