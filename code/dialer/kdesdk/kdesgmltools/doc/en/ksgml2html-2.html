2. Use

Go to the directory where your SGML file is. Type ksgml2html filename

You will be asked for the Description and the keywords.

Type in a short description for your document, and the related keywords.

These data will be included in the resulting HTML files as META tags. They are used by numerous search engines like AltaVista to classify and query HTML pages.

The description and the keywords will be saved with other neccessary data in a file named after your sgml document with the extention "nif". The next time you use ksgml2html for this document you will be informed that ..existing nif file is used. If you want to change the description or the keywords, just remove this "nif" file and run ksgml2html again.
