1. General

ksgml2html is part of the kdesgmltools package. kdesgmltools allows the user to create good-looking HTML output from sgml (linuxdoc.dtd) files. There are many things you can do with kdesgmltools, but if you just want the HTML output to have the KDE look, just use ksgml2html.
