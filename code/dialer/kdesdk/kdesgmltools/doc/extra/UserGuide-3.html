3. What you get

From version 0.5 many things have changed. So, even if you were familiar with older versions, read this chapter.

When you originally wanted to create HTML from SGML you used the command 

sgml2html sgmlfile.sgml
Now you have a new option invoced with --nice or -N.

Typing 

sgml2html --nice sgmlfile.sgml
activates the kdesgmltools modifications. I strongly recomend using --nice in association with --imagebuttons (same as -I).

You also get a new mapping from sgml to html called "nice" too :-)
