2. Installation

The current version of kdesgmltools is v0.6 . This version works with sgml-tools-1.0.0 and MAY NOT WORK WITH OLDER VERSIONS.

The installation is now automated.

Create an empty directory and move there kdesgmltools-0.6.tar.gz. Extract the contents with 

gzip -dc kdesgmltools-0.5.tar.gz | tar xv

Type: 

# ./configure
# make install
