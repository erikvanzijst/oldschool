7. Especially for kde documentation.

If you want your resulting HTML file to have the "kde look", a small script called ksgml2html is included in the package.

ksgml2html creates an appropriate niceInclude file (nif) in the directory where it is invoced from and runs sgml2html with the right options. When you run it you will be asked for the "Description" and the "keywords". The answers will form two META tags in the resulting HTML document, useful for search engines.
