lecturer: Jeroen Ketema.

Book

Gerard Tel, Introduction to Distributed Algorithms, Cambridge University Press, 2000 (2nd edition).

Goal

To obtain a good understanding of a large range of distributed algorithms, with an emphasis on algorithms that are of importance for the course Distributed Systems.

Prerequisites

Data Structures (400145), Computer Networks (400016).

Topics

Transition systems, routing algorithms, deadlock-free packet switching, wave and traversal algorithms, election algorithms, termination detection, anonymous networks, synchronous networks, fault tolerance, failure detection.

Schedule per week

  1. H 1, 2: Introduction: Distributed systems; The model: Transition systems, assertions, causal order.
  2. H 4.2, 4.3.1: Routing: Floyd-Warshall algorithm, Toueg's algorithm, Chandy-Misra algorithm, Netchange algorithm, Merlin-Segall algorithm.
  3. H 5.1, 5.2, 5.3.1, 6.1.1, 6.3.4, 6.4: Deadlock-free packet switching: Destination scheme, hops-so-far scheme, acyclic orientation covers, forward-count controller; Traversal algorithms: Wave algorithms, distributed depth-first search, Awerbuch's algorithm, Cidon's algorithm.
  4. H 7.1, 7.2.1, 7.2.2, 6.2.2, 6.2.3, 7.3.1-7.3.4: Leader election: Chang-Roberts algorithm, Franklin's algorithm, Dolev-Klawe-Rodeh algorithm, tree algorithm, echo algorithm with extinction, Gallager-Humblet-Spira algorithm.
  5. H 8.1.1, 8.2, 8.3.1, 8.3.2, 9.1, 9.2.1, 9.3: Termination detection: Dijkstra-Scholten algorithm, Shavit-Francez algorithm, Dijkstra-Feijen-van Gasteren algorithm, Safra's algorithm; Anonymous networks: Probabilistic algorithms, Itai-Rodeh election algorithm, election for arbitrary networks, FireWire.
  6. H 9.4: Computing network size, Itai-Rodeh ring size algorithm.
  7. H 10.1, 10.2, 10.3.1, 12.4.3: Snapshots: Chandy-Lamport algorithm, Lai-Yang algorithm; Breadth-first search: Frederickson's algorithm (simple and advanced version).
  8. H 13.1, 13.2.1-13.2.3, 14.1, 14.4: Fault tolerance: Impossibility of 1-crash consensus, Bracha-Toueg t-crash consensus algorithm.
  9. H 14.4, 16.1-16.4: Bracha-Toueg t-Byzantine consensus algorithm; Failure detection: Implementation of failure detection, weakly accurate failure detector, Chandra-Toueg failure detector.
  10. Good Friday (University closed.)
  11. H 17.1.1, 17.1.3: Stabilization: Dijkstra's token ring. (Last lecture, will last only 45 minutes.)
